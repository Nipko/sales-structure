## Flow must be:
1. Onboarding → user selects template → first agent auto-created with that template
2. Enter "AI Agent" page → agent is ALREADY there, configured from template
3. Banner says "Finish personalizing your agent" (NOT "configure from scratch")
4. User can create MORE agents with templates (plan-gated)
5. Templates must be VERY complete — powerful, skilled, well-researched for each type

**Why:** User expects continuity from onboarding to dashboard. The agent should feel ready to use, not empty.
**How to apply:** Banner checks if agent has default template name (unpersonalized) not if it's unconfigured. Templates must include comprehensive rules, behaviors, greetings, tools.
